## 🚀 Main Features

### 🛡️ Privacy
- **Ghost Mode:** Stay invisible with hidden blue ticks, typing indicators, and last seen.
- **Anti-Revoke:** Prevent messages and statuses from being deleted for you.
- **Freeze Last Seen:** Keep your last seen timestamp static.
- **Custom Privacy:** Set specific privacy rules for individual contacts.
- **Hide View Once & Audio Seen:** Open "View Once" media multiple times without notifying the sender.
- **Show Edited Message History:** See the original content of edited messages.

### 🎨 Customization
- **Theme Manager:** Apply custom CSS themes to change the entire look of WhatsApp.
- **Material You (Monet):** Dynamic coloring support based on your system wallpaper.
- **Custom Colors:** Personalize primary, secondary, background, and bubble colors.
- **Home Screen Tweaks:** Separate groups from chats, hide tabs, and customize status styles (IG Style, Vertical, etc.).
- **List Animations:** Add smooth animations when scrolling through your chat list.

### 📁 Media & Utilities
- **Download Everything:** Easily save statuses and "View Once" media.
- **High Quality Media:** Send images and videos in HD, 60FPS, and original resolution.
- **Call Recording:** Record incoming and outgoing calls automatically.
- **Audio Transcription:** Native WhatsApp audio transcription support.
- **Tasker Automation:** Use intents to automate WhatsApp actions via Tasker.
- **Bootloader Spoofer:** Bypass unofficial app checks on rooted devices.

### 🛠️ General Tweaks
- **Increased Limits:** Up to 60 pinned chats and removed forward limits.
- **DND Mode:** Quickly disable WhatsApp internet access with a dedicated button.
- **Online Toasts:** Get notified immediately when a contact comes online.
- **Translate Integration:** Built-in Google Translate for effortless communication.

## 📥 Installation

1. **Root Access:** Ensure your device is rooted.
2. **Xposed Framework:** Install [LSPosed](https://github.com/JingMatrix/LSPosed) (recommended).
3. **Download:** Get the latest Whatsapp Toolkit APK.
4. **Install:** Install the APK on your device.
5. **Enable:** Open LSPosed Manager, enable the **Whatsapp Toolkit** module, and select WhatsApp as the scope.
6. **Reboot:** Restart WhatsApp to apply the changes.

## ⚙️ Configuration

The module settings can be accessed directly from the **Whatsapp Toolkit** app icon in your launcher. Use the **Backup/Restore** feature in the Home tab to save your configurations.
