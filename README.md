## 🚀 Key Features

### 🛡️ Privacy & Stealth
- **Ghost Mode (Extended):** Hide blue ticks (send only on reply), typing indicators, and recording status.
- **Anti-Revoke 2.0:** Prevent messages, media, and statuses from being deleted for everyone. Includes a built-in deleted message browser.
- **Freeze Last Seen:** Keep your timestamp static while staying active.
- **Granular Custom Privacy:** Configure specific rules per contact or group.
- **Stealth Media:** View "View Once" content unlimited times and listen to audio without sending "Played" receipts.
- **Archive Cloaking:** Choose different behaviors for the Archived chats entry.

### 🎨 Theming & Interface
- **Dynamic Monet Engine:** Full Material You support that adapts to your system wallpaper.
- **Emerald Color Presets:** Quick-select professional color palettes (Blue, Cyan, Pink, etc.).
- **Bubble Lab:** Customize incoming and outgoing message bubble colors with a hex picker.
- **Architecture Tweaks:** Separate groups into a dedicated tab, hide navigation tabs, and customize the Home Toolbar.
- **Facebook-Style Status:** Modern horizontal status bar on the home screen with customizable UI styles.

### 📁 Media God-Mode
- **Lossless Uploader:** Bypasses WhatsApp's aggressive compression for true HD/4K status and media uploads.
- **Pro Media Tools:** Support for 60FPS video, original resolution photos, and increased file size limits (up to 2GB).
- **Auto-Download:** Automatically save "View Once" media and statuses to your gallery.
- **Voice Mastery:** Built-in **Voice Note Transcription** and real-time **Voice Changer** with pitch effects.

### 🛠️ Advanced Tools
- **Bootloader Spoofer Pro:** Advanced bypass for unofficial app detection on rooted devices. Now supports **Custom Keybox XML** importing for personalized attestation keys.
- **AI Suite:** Smart Smart Replies, Message Summarization (TL;DR for busy groups), and built-in Google Translate.
- **Call Management:** Complete **Call Blocker** and an improved **Call Recording** system with a dedicated management UI.
- **Power User Tweaks:** Remove the 3-chat pin limit (up to 60+), remove forward limits, and enable "Always Online."

## 📱 Technical Support
- **Framework:** LibXposed API 101 (Modern) & Legacy LSPosed (Bridge).
- **OS Support:** Android 9.0 through Android 15+.
- **WhatsApp Versions:** Optimized for standard WhatsApp (com.whatsapp) and Business (com.whatsapp.w4b) versions **2.26.x.xx**.

## 📥 Installation

1. **Root Access:** Your device must be rooted.
2. **Xposed Framework:** Install [LSPosed](https://github.com/JingMatrix/LSPosed) (Recommended).
3. **Module Scope:** Enable **Whatsapp Toolkit** in LSPosed Manager and select **WhatsApp**, **WhatsApp Business**, and **Android System** in the scope.
4. **Initialization:** **Restart WhatsApp twice** after the first install to allow the DexKit discovery engine to map the latest obfuscated methods.
